## Installation

```sh
npm install react-native-animated-image-loader
```

## Usage

```js
import AnimatedImgLoader from 'react-native-animated-image-loader';

...
<AnimatedImgLoader
  width={100}
  height={100}
  imageUri={'https://images.pexels.com/photos/1366919/pexels-photo-1366919.jpeg'}
/>
```
